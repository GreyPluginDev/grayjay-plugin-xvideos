Basic Spankbang Plugin for Grayjay the app

Basic Workflow working fine

Try it

GrayJay -> Sources -> +
And scan this QR code

![qr-code (1)](https://github.com/ruiaso/grayjay-plugin-spankbang/blob/test/assets/SBTest.png)

What's working:


    Video download (HLS)
    Video search
    Video playback
    Video info

What's not working:


    Channels search
    Channel subscribe/unsubscribe
    Channel info fetch (description, about, its videos)
    View profiles for models, pornstars, and channels
    Fetching video comments with like/dislike, avatar, etc. (only non-nested ones)
    Subtitles
    Video quality selection
    Per-channel video search
    Search filters
    Premium login
    ...all the rest
